# Library-Database-Schema
Library management system database schema using MySQL Workbench.

## 🏗️ Database Overview

### Tables:
- 📚 `Books`
- 👤 `Authors`
- 🏷 `Categories`
- 📘 `BookAuthors` (many-to-many)
- 🙋‍♂️ `Members`
- 🔄 `BorrowRecords`
